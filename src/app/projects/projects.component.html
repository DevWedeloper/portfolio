This is a dashboard simulator that has advanced CRUD interaction, I made a script to generate my own data to play around with.

It has features like authorization, accessibility limits depending on user role, caching, and many more!

There are two roles, standard and admin. Standard users can only do GET actions, while admin users can do CREATE, UPDATE, DELETE actions.

To get started, use either of these accounts:

username & password: admin

username & password: standard
